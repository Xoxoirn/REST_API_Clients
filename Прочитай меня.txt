REST API сделан с использованием Vue.js и Bootstrap
Также я использовала nodemon для того, чтобы заново не перезапускать сервер после внесения изменений в код
Кнопка "Выделить" позволяет просто выделить клиента красным цветом (изменение поля marked у клиента) 
Кольцо загрузки просто для красоты
Клиенты хранятся в файле, при выключении сервера новые данные сохраняются
